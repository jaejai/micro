Features

A five-stage pipeline that takes a raw EBSD scan all the way to a PowerPoint report.

Stage 1

Load & Resample

Read TSL .ang scans and resample the hexagonal measurement grid to a square grid for downstream analysis.

Stage 2

Microstructure

Compute IQ, CI, and IPF maps with grain-boundary detection, plus optional low-CI noise cleanup for accurate maps.

Stage 3

Grain Size

Segment grains with union-find and measure grain size following the ASTM E2627 standard.

Stage 4

Texture (ODF)

Build a GSH-based orientation distribution function with φ₂ sections and α/γ fiber plots.

Stage 5

Report

Export every figure and result table to a formatted PowerPoint deck in a single click.

Bonus

Zero setup

One installer builds a self-contained environment from conda-forge — no Python or conda knowledge required.
